## 计算设备

GenieX 在骁龙 X 系列计算平台上运行，支持在 Windows on Arm 笔记本和台式机上进行本地 AI 推理。

### 支持的芯片组
- Snapdragon X Elite
- Snapdragon X2 Elite

### 开始使用

**方式 1：物理设备**

使用骁龙 X 系列笔记本或台式机（例如搭载 Snapdragon X Elite 的 Copilot+ PC）。安装 GenieX 并直接运行推理。


**方式 2：高通开发者云**

在 [Qualcomm Developer Cloud - Compute 页面](https://qdc.qualcomm.com/devices/compute) 点击 "Compute"，打开 `Snapdragon X Elite` 或 `Snapdragon X2 Elite` 设备。

<Tip>
**SSH 密钥权限错误？** 如果连接时出现 `WARNING: UNPROTECTED PRIVATE KEY FILE!`，说明 `.pem` 文件权限过于开放。执行以下命令修复：

```bash
chmod 600 /path/to/your-key.pem
```

SSH 要求私钥仅对所有者可读（权限 `600`）。
</Tip>
