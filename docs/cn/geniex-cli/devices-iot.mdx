## IoT 设备

GenieX 支持高通 IoT 平台，用于智能摄像头、机器人、工业自动化和嵌入式系统的边缘 AI 推理。

### 支持的芯片组
- Qualcomm QCS9075
- Qualcomm QCS9100
- Qualcomm QCS8275

### 开始使用

**方式 1：物理设备**

使用高通 IoT 开发套件（例如 Dragonwing RB3 Gen 2 或 QCS8550 HDK）。在 Linux Arm64 镜像上安装 GenieX 并运行推理。


**方式 2：高通开发者云**

在 [Qualcomm Developer Cloud - IOT 页面](https://qdc.qualcomm.com/devices/iot) 点击 "IOT"，打开 `Qualcomm Dragonwing IQ-9075` 或 `Qualcomm Dragonwing IQ-9100 Beta` 设备。

<Tip>
**SSH 密钥权限错误？** 如果连接时出现 `WARNING: UNPROTECTED PRIVATE KEY FILE!`，说明 `.pem` 文件权限过于开放。执行以下命令修复：

```bash
chmod 600 /path/to/your-key.pem
```

SSH 要求私钥仅对所有者可读（权限 `600`）。
</Tip>
