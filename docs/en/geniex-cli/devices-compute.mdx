## Compute Devices

GenieX runs on Snapdragon X-series compute platforms, enabling local AI inference on Windows on Arm laptops and desktops.

### Supported Chipsets
- Snapdragon X Elite
- Snapdragon X2 Elite

### Get Started

**Option 1: Physical Device**

Use a Snapdragon X-series laptop or desktop (e.g., a Copilot+ PC with Snapdragon X Elite). Install GenieX and run inference directly.


**Option 2: Qualcomm Developer Cloud**

Click "Compute" at the [Qualcomm Developer Cloud - Compute page](https://qdc.qualcomm.com/devices/compute) and open a `Snapdragon X Elite` or `Snapdragon X2 Elite` device.
