## Device Categories

GenieX supports on-device AI inference across four Qualcomm Snapdragon device categories:

| Category | Chipsets | Use Cases |
|----------|----------|-----------|
| **Compute** | Snapdragon X Elite, X2 Elite | Windows on Arm laptops, desktops |
| **Mobile** | Snapdragon 8 Elite, 8 Elite Gen 5 | Android flagship smartphones |
| **IoT** | Qualcomm IQ9, IQ8 | Edge devices, smart cameras, robotics |
| **Automotive** | Qualcomm SA8775P, SA8295P, SA7255P | In-vehicle infotainment, ADAS |

## Chipset Coverage

Each chipset exposes the Qualcomm Hexagon NPU for accelerated inference. GenieX automatically detects the available hardware and selects the optimal execution path.

### How Chipset Support Works

1. **NPU Detection** — GenieX queries the Hexagon DSP/HTP capabilities at runtime.
2. **Model Compatibility** — Models compiled via QAIRT target specific HTP architectures. Not every model runs on every chipset.
3. **Fallback** — If NPU execution is unavailable, GenieX falls back to CPU (GGUF plugin) where supported.

<Note>
Chipset support depends on the QAIRT SDK version bundled with your GenieX installation. Newer chipsets may require updating to the latest GenieX release.
</Note>
